Documentation

This is a simple API that allows users to view, create, update, and delete movies. Use the following endpoints:

Method Endpoint Description Request Body Response Body
GET /movies Returns all movies in the database None A list of all movies in the database in a JSON format.
GET /movies/[title] Returns a single movie None A single movie in the database in a JSON format.
GET /movies/genre/[genre] Returns all movies in a single genre None A list of all movies of the selected genre in the database in a JSON format.
GET /movies/director/[director] Returns all movies by a single director None A list of all movies by the selected director in the database in a JSON format.
GET /users Returns all users in the database None A list of all users in the database in a JSON format.
GET /users/[username] Returns a single user None A single user in the database in a JSON format.
POST /movies Add to the list of movies { "Title": "[Enter title here]", "Director": "[Enter director here]", "Genre": "[Enter primary genre here]", "Year": "[Enter year of release here]" } A JSON object with the movie added to the list
POST /users Register a new user {"Username": "[Enter name here]", "Password": "[Enter password here]", "Email": "[Enter email here]", "Birthday": "[Enter as a date (YYYY-MM-DD)]" } A JSON object with the user entered
POST /users/[username]/movies/[movieid] Add a movie to your list None Movie added to your list.
PUT /users/[username] Update a user's information {"[Put object to update here]": "[Enter updated information here]" } User updated.
DELETE /[username]/movies/[movieid] Delete a movie from your list None Movie deleted.
DELETE /user[username] Delete a user None User deleted.

For full API and code documentation, see the JSDoc documentation.
